# Retail Order Service API

Сервис автоматизации закупок в розничной сети. REST API для управления товарами, заказами и поставщиками.

## Технологии

- Python 3.10+
- Django 4.2
- Django REST Framework 3.14
- SQLite (разработка) / PostgreSQL (продакшн)
- JWT Token аутентификация
- Celery (для асинхронных задач)
- Redis (брокер сообщений)
- drf-spectacular (документация OpenAPI)

## Функциональность

### Пользователи
- ✅ Регистрация с подтверждением email
- ✅ Авторизация по токену
- ✅ Восстановление пароля
- ✅ Управление профилем
- ✅ Добавление/редактирование контактов доставки

### Товары
- ✅ Просмотр каталога с фильтрацией
- ✅ Поиск по названию и артикулу
- ✅ Детальная информация о товаре
- ✅ Категории товаров
- ✅ Динамические параметры товаров

### Заказы
- ✅ Работа с корзиной (добавление/удаление)
- ✅ Оформление заказов
- ✅ Подтверждение заказа с email уведомлением
- ✅ Отслеживание статуса заказа
- ✅ История заказов

### Для поставщиков
- ✅ Управление прайс-листом
- ✅ Импорт товаров (YAML/CSV)
- ✅ Включение/отключение приема заказов
- ✅ Просмотр заказов с товарами поставщика
