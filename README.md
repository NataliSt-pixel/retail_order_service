# Retail Order Service

Сервис автоматизации закупок в розничной сети.

## Функциональность

### Пользователи
- Регистрация и авторизация
- Управление профилем
- Управление контактами доставки

### Товары
- Просмотр каталога товаров
- Фильтрация и поиск
- Импорт товаров (для поставщиков)

### Заказы
- Работа с корзиной
- Оформление заказов
- Отслеживание статуса заказов
- Email уведомления

### Для поставщиков
- Управление прайс-листом
- Включение/отключение приема заказов
- Просмотр заказов
